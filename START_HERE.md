### Model E2E: ML Model Download
```powershell
python -m jessica --demo model-e2e
```
**What happens:**
1. Detects "sentiment analysis" task
2. Suggests DistilBERT model

4. Loads into transformers
5. Runs inference
6. Caches for instant reuse

**Result:** Real AI model loaded and tested (59,138 parameters)

**Tests fail?**

