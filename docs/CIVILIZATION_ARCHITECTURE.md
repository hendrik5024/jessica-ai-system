## How Civilization Fits In

### Position in Stack
The **Internal Civilization** sits **above the Agent Loop**.

This is crucial:
- Agent Loop generates draft response
- **Civilization reviews it from 6 angles**
- Judge arbitrates, making final decision
- Response returned to user

### Decision Flow Example

**User asks:** "Should I leave my job?"

```
1. AGENT LOOP generates draft
   → "Career change depends on your priorities..."

2. CIVILIZATION CONVENES
   ├─ Strategist: "5-10 year analysis shows value creation"
   ├─ Human Advocate: "Your autonomy, your choice"
   ├─ Risk Sentinel: "Job security gone, tail risk 0.65"
   ├─ Explorer: "Novel opportunity, explore further"
   ├─ Archivist: "Consistent with your risk patterns"
   └─ Judge: "All perspectives noted. No constitutional issues."

3. OBJECTIONS RAISED
   └─ Risk Sentinel → Explorer: "Overconfident in market assumptions"

4. ARBITRATION
   └─ Judge: "Approved with dissent. Risk Sentinel recorded."

5. INSTITUTIONAL MEMORY
   └─ Session stored: All viewpoints, objections, precedent

6. RESPONSE WITH TRANSPARENCY
   → Draft + Judge's synthesis + Risk Sentinel's concern
```
